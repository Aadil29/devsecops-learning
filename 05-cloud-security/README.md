# Cloud Security

Find and fix real misconfigurations in AWS.

## What to build

- Use AWS free tier (or an existing account) — don't simulate this
- Run ScoutSuite or Prowler against the account and read the output
- Fix at least 3 real findings (overly permissive IAM policy, public bucket, unencrypted resource, etc.)
- Set up CloudTrail if not already on

## Done when

- [ ] Audit tool ran and produced a real report
- [ ] At least 3 findings fixed and re-scanned to confirm
- [ ] Notes explain what each finding meant and why it matters
- [ ] Notes written up
