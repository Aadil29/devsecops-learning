# Containers

Build and secure a containerised app.

## What to build

- A simple web app (any language) with an API endpoint, a database, and user input
- A `Dockerfile` and `docker-compose.yml` to run it
- Intentionally leave at least one insecurity in (root user, hardcoded secret, etc.)
- Scan it with Trivy and note what it finds

## Done when

- [ ] `docker-compose up` runs the app
- [ ] Trivy scan runs locally and finds at least one issue
- [ ] Notes written up
